## 1. 글로벌 벤치마킹 (상세)

### 1-1. FeelinMySkin — 가장 참고해야 할 모델 ⭐⭐⭐

부부 개발자 2명이 만든 인디 앱. BARDA와 가장 유사한 포지션이면서 "서비스 구색"이 가장 잘 갖춰져 있음.

| 항목 | 내용 |
|---|---|
| 운영사 | Andras Munkacsi (부부 개발, 헝가리) |
| 제품 DB | 15만개+ |
| 플랫폼 | iOS + Android + Chrome 확장 |
| 수익모델 | 프리미엄 구독 (PRO) |
| 핵심 강점 | "매일 돌아오게 만드는" 트래킹 구조 |

**기능 구성:**

| 기능 | 설명 | BARDA 시사점 |
|---|---|---|
| 루틴 플래너 | AM/PM 루틴 설정 + Smart Ordering(자동 순서 배치) + 체크리스트 | BARDA도 분석 후 "매일 체크리스트"로 전환 필요 |
| 성분 분석기 | INCI 성분 체커 + 피부 고민별 매칭 점수 | 성분사전과 연동하면 유사 구현 가능 |
| 제품 트래커 | 보유 제품 목록 + 사용량 + 유통기한 + 가격 추적 | "내 서랍" 기능으로 대응 |
| 스킨 다이어리 | 매일 사진 + 수면/기분/운동 로그 + 피부 변화 추적 + Analytics | 리텐션 핵심. BARDA도 다이어리 필요 |
| 커뮤니티 포럼 | 질문/답변 + 루틴 공유 + 스터디 허브(교육 콘텐츠) | BARDA Q&A + 피드와 유사 |
| 제품 스캐너 | 사진 찍으면 DB에서 제품 검색 + Skin Match Score | Phase 3 기능 |
| 루틴 추천 | 피부 고민 기반 맞춤 루틴 템플릿 자동 생성 | BARDA 분석 결과와 연결 가능 |

**FeelinMySkin에서 배울 핵심:**
- "분석"은 입구이고, "매일 트래킹"이 서비스의 본체
- 체크리스트를 매일 열게 만들면 DAU가 생기고, DAU가 있어야 커뮤니티가 살아남
- 교육 콘텐츠(Study Hub)가 신뢰감을 만들고 SEO 유입 채널이 됨

### 1-2. SkinSort — DB + 커뮤니티 루틴 공유

| 항목 | 내용 |
|---|---|
| 운영사 | SkinSort (미국) |
| 제품 DB | 대규모 (글로벌 제품) |
| 플랫폼 | 웹 + iOS + Android |
| 수익모델 | 프리미엄 구독 |
| 핵심 강점 | 제품 DB 깊이 + 듀프 파인더 + 루틴 퍼블리시 |

**기능 구성:**

| 기능 | 설명 | BARDA 시사점 |
|---|---|---|
| 루틴 빌더 | 제품 추가 → 자동 순서 + 충돌 감지 + 초보자 가이드 | BARDA 루틴 분석과 동일 |
| 제품 개인화 점수 | 피부타입 + 성분 + 과거 좋아요 블렌딩 → 제품별 스마트 점수 | Phase 3에서 구현 가능 |
| 듀프 파인더 | 비싼 제품의 저렴한 대안 찾기 | 사용자 니즈 높음, 향후 킬러 피처 후보 |
| 루틴 퍼블리시 | 내 루틴을 커뮤니티에 공개 + 링크로 어디서든 공유 | BARDA 피드의 핵심 기능과 일치 |
| 성분 브레이크다운 | 모든 제품의 성분을 쉽게 이해할 수 있는 분석 | 성분사전 + 제품 상세 페이지 |

**SkinSort에서 배울 핵심:**
- "루틴 퍼블리시 + 링크 공유"가 바이럴의 핵심 메커니즘
- 듀프 파인더는 사용자 검색 의도가 매우 높은 기능 (SEO 가치 높음)
- 제품 DB의 깊이가 곧 서비스의 신뢰도

### 1-3. Skin Bliss — AI 기술 풀스택

| 항목 | 내용 |
|---|---|
| 운영사 | Skin Bliss (미국/유럽) |
| 플랫폼 | iOS + Android |
| 수익모델 | 프리미엄 구독 (무료 체험 후) |
| 핵심 강점 | AI 기술 차별화 (얼굴 스캔, 사진 비교, 선반 인식) |
| 실적 | 84% 사용자 피부 개선 경험 |

**기능 구성:**

| 기능 | 설명 | BARDA 시사점 |
|---|---|---|
| AI 얼굴 스캔 | 사진 → 피부 프로필(타입 + 고민) 자동 진단 | Phase 3 AI 기능 |
| 루틴 빌더 + 타임라인 | 루틴이 목표 달성에 맞는지 평가 + "몇 주 후 효과 예상" | 차별화 포인트: 기대 효과 타임라인 |
| 루틴 플레이어 | 미러모드로 단계별 안내 + 타이머 | UX 아이디어 참고 |
| AI 사진 비교 | 매일 사진 → AI가 미세한 변화 분석 | Phase 3 프리미엄 |
| 선반 분석 | 선반 사진 한 장 → 제품 일괄 인식 + 분석 | Phase 3 제품 스캐너 확장 |
| 날씨 기반 루틴 | 오늘 기후/UV/습도 기반 제품 추천 | 간단 구현 가능, 차별화 포인트 |
| 14개 언어 지원 | 한국어 포함 | 글로벌 진출 시 경쟁 대상 |

**Skin Bliss에서 배울 핵심:**
- "루틴 타임라인"(몇 주 후 효과 예상)이 사용자 동기부여에 매우 효과적
- AI 기능은 비싸지만, 프리미엄 구독의 정당성을 만들어줌
- 날씨 기반 루틴은 API 하나로 간단히 차별화 가능

### 1-4. Skincare Routine App — 심플 + 원타임 결제

| 항목 | 내용 |
|---|---|
| 운영사 | Mento Apps (영국) |
| 제품 DB | DECIEM(The Ordinary/NIOD) 중심 + 커스텀 추가 |
| 플랫폼 | iOS + Android |
| 수익모델 | 원타임 결제 (구독 없음) |
| 핵심 강점 | 심플함 + 충성 사용자 |
| 언어 | 16개 언어 (한국어 포함) |

**기능 구성:**

| 기능 | 설명 | BARDA 시사점 |
|---|---|---|
| AM/PM 루틴 자동 배치 | 제품 추가 → 자동 순서 + 시간대 배치 | BARDA와 동일 |
| 충돌 감지 + 무시 옵션 | 충돌 경고 + "알고 있어, 무시할래" 선택 | 사용자 자율성 존중 (BARDA에 적용 가능) |
| 요일별 스케줄링 | 특정 요일/날짜/N일 간격으로 제품 배치 | BARDA 7일 캘린더와 유사하나 더 세밀 |
| 사용 추적 + 체크오프 | 매일 아침/저녁 체크 + 최근 6일 사용 이력 | "매일 체크리스트" 참고 |
| 피부 다이어리 | 특정 날 피부 상태 메모 (좋음/나쁨 + 노트) | 간단한 다이어리 모델 |
| 제품 가격/용량 추적 | ROI 계산 ("이 제품 1회당 얼마") | 제품 트래커 차별화 아이디어 |

**Skincare Routine App에서 배울 핵심:**
- 원타임 결제 모델이 진입장벽을 낮추고 충성도를 만듦 (BARDA 9,900원 모델과 일치)
- "충돌 무시" 옵션이 사용자 자율성을 존중 → UX 개선 포인트
- 심플함이 강점. 기능이 많아도 복잡하면 안 됨

### BARDA만의 차별점 (이미 있는 것)
- ✨ **루틴 점수 시스템**: 다른 앱에 없음. 바이럴 핵심
- ✨ **7일 자동 캘린더**: 레티놀/각질 자동 분배는 BARDA만의 강점
- ✨ **AM/PM 맞춤 TIP**: 피부타입×시간대별 가이드
- ✨ **K-뷰티 특화**: 한국 제품 DB + 한국어 UX
- ✨ **충돌 무시 불가**: "왜 안 되는지" 교육적 접근

### 핵심 갭 요약
> **문제: "한 번 분석하고 끝"**
> 분석은 1회성이지만, 서비스는 매일 돌아와야 산다.
> FeelinMySkin/Skin Bliss/Skincare Routine App 모두 **"매일 체크리스트 + 다이어리"**로 리텐션을 잡고 있음.

## 3. 이사님 제안 기능 검토 (A/B/C/D)

### A. 홈 화면 (랜딩 페이지) — 최우선 ✅

| 섹션 | 내용 | 검토 의견 |
|---|---|---|
| 히어로 | 서비스 한줄 소개 + "내 루틴 분석하기" CTA | ✅ 좋음. 단, **로그인 사용자는 "오늘의 루틴 체크리스트"가 홈**이어야 함 (벤치마킹 핵심 인사이트) |
| 핵심 기능 3가지 | 성분 충돌 분석 / AM·PM 루틴 제안 / 7일 캘린더 | ✅ 좋음. 아이콘 + 한줄 설명으로 직관적으로 |
| 실시간 피드 미리보기 | 최근 공유된 루틴 3~4개 카드 → 피드 이동 | ✅ 좋음. "살아있는 서비스" 느낌. 초기에 더미 데이터라도 넣어두기 |
| 통계 | "지금까지 N개 루틴 분석됨" 등 신뢰감 수치 | ✅ 좋음. 소셜 프루프 효과. 초기엔 실제 수치 + 약간 과장 없이 |

**추가 제안:**

| 추가 섹션 | 내용 | 이유 |
|---|---|---|
| 인기 루틴 TOP 3 | "건성 피부 인기 루틴 1위" 카드 | 탐색 욕구 자극 + "나도 해봐야지" |
| 가이드 CTA | "레티놀 초보자 가이드 →" 같은 콘텐츠 링크 | SEO 유입 랜딩 포인트 + 신뢰감 |

**홈 화면 상태 분기 제안:**

```
비로그인 사용자:
  히어로 → 핵심 기능 3가지 → 피드 미리보기 → 통계 → "시작하기" CTA

로그인 사용자 (분석 완료):
  오늘의 루틴 체크리스트 (AM/PM) → 피부 컨디션 간단 체크 → 피드 새글 → 내 루틴 점수

로그인 사용자 (분석 전):
  "내 루틴 분석하기" CTA → 핵심 기능 소개 → 피드 미리보기
```

> 💡 **핵심 포인트**: 비로그인 = 마케팅 랜딩, 로그인 = 매일 쓰는 도구. 이 분기가 리텐션의 핵심.

### B. 커뮤니티 강화 — 높음 ✅

| 기능 | 내용 | 검토 의견 |
|---|---|---|
| 댓글 시스템 | 루틴 포스트에 댓글 입력/조회 | ✅ 필수. API 있으면 UI만 붙이면 됨. 댓글 좋아요도 같이 |
| 마이페이지 완성 | "내 분석 히스토리" + "좋아요한 글" 실제 데이터 연동 | ✅ 필수. 여기에 **"저장한 루틴"** 탭도 추가 |
| 피드 검색 | 피부타입·고민·제품명으로 루틴 검색 | ✅ 좋음. 필터 UI가 핵심. "나와 같은 건성+보습" 원터치 필터 |
| 루틴 상세 페이지 | `/feed/[id]` — 전체 루틴 + 댓글 스레드 | ✅ 필수. **공유 링크의 랜딩 페이지**이기도 함 (SEO + 바이럴) |

**추가 제안:**

| 추가 기능 | 내용 | 우선순위 |
|---|---|---|
| "이 루틴 따라하기" 버튼 | 다른 사람 루틴의 제품을 내 분석에 자동 입력 | 🔴 높음 (SkinSort 핵심 기능) |
| 루틴 공유 링크 | `/feed/[id]`를 카카오톡/인스타로 공유 (OG 이미지 자동 생성) | 🔴 높음 (바이럴 핵심) |
| BARDA 공식 가이드 태그 | Q&A에 성분 키워드 감지 → 자동 가이드 표시 | 🟡 중간 (신뢰도↑) |

### C. 참여·리텐션 기능 — 중간 ✅ (일부 상향 조정 필요)

| 기능 | 내용 | 검토 의견 | 우선순위 조정 |
|---|---|---|---|
| 프로필 설정 | 닉네임, 피부타입 배지, 프로필 이미지 | ✅ 좋음. 피부타입 배지가 커뮤니티에서 맥락을 줌 | 중간 유지 |
| 루틴 저장/비교 | 내 과거 분석 결과 저장 & 비교 | ✅ 좋음. "3개월 전 vs 지금" 비교가 동기부여 | 중간 유지 |
| 제품 리뷰 | 제품별 별점·한줄평 (제품 DB 활용) | ⚠️ **주의 필요**. 화해와 정면 경쟁이 됨. "루틴 안에서의 제품 경험"으로 차별화 필요 | 중간 유지, 접근 변경 |
| 알림 | 내 글에 좋아요/댓글 알림 | ✅ 좋음. 리텐션 기본 | 중간 유지 |

**제품 리뷰 차별화 방향:**

```
화해 리뷰: "이 제품 보습력 좋아요" (제품 단독 평가)
    ↕ 차별화
BARDA 리뷰: "건성+레티놀 루틴에서 이 크림 쓰니까 자극 없이 좋았어요" (조합 맥락 리뷰)
```

> 제품 리뷰를 받되, 반드시 **"어떤 루틴에서 썼는지"** 맥락이 붙어야 함. 이래야 화해와 겹치지 않고 BARDA만의 데이터가 됨.

**추가 제안 — 🔴 상향 조정 필요:**

| 추가 기능 | 내용 | 우선순위 | 이유 |
|---|---|---|---|
| **매일 루틴 체크리스트** | 분석 완료 후 매일 AM/PM 제품 체크오프 | 🔴 **최우선** | 벤치마킹 결과 모든 성공 서비스의 공통 핵심 |
| **스킨 다이어리 (간단 버전)** | 매일 피부 컨디션(😊😐😣) + 한줄 메모 | 🔴 **높음** | 체크리스트와 세트. 데이터 축적 → 패턴 분석 |
| **내 서랍 (제품 관리)** | 보유 제품 목록 + 개봉일 + "다 썼어요" | 🟡 중간 | 루틴 분석 시 등록한 제품 자동 연동 |

## 6. 핵심 인사이트 요약

1. **"분석 도구"에서 "매일 쓰는 루틴 앱"으로 전환**해야 함. 분석은 입구, 체크리스트가 본체.
2. **홈 화면 분기**(비로그인: 랜딩 / 로그인: 체크리스트)가 서비스 구색의 핵심 변화.
3. **매일 체크리스트 + 간단 다이어리**가 리텐션의 최소 조건. 이게 없으면 커뮤니티도 안 살아남.
4. **성분 가이드는 "나중"이 아니라 SEO 유입 채널**. 핵심 10개만이라도 빨리 만들 것.
5. **제품 리뷰는 "조합 맥락 리뷰"**로 차별화. 화해식 단독 리뷰와 겹치면 안 됨.
6. **BARDA만의 우위**(루틴 점수, 7일 캘린더, AM/PM TIP)는 이미 강력. 여기에 트래킹을 더하면 경쟁력 확보.
