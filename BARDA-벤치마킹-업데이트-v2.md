## 0. 요약 — 한국 시장의 빈 칸

> **"내가 쓰는 여러 제품, 같이 써도 되는 걸까?"**
> 이 질문에 답하는 한국 서비스가 **아직 없다.**

| 서비스 | 성분 분석 | 조합 충돌 분석 | 한국 제품 DB | 한국어 |
|--------|----------|---------------|-------------|--------|
| 화해 | ✅ 개별 성분 | ❌ 단독 제품만 | ✅ | ✅ |
| 글로우픽 | ✅ 리뷰 중심 | ❌ | ✅ | ✅ |
| FeelinMySkin | ✅ INCI | ❌ | ❌ 글로벌 | ❌ |
| SkinSort | ✅ 성분 점수 | ✅ 충돌 감지 | ❌ 글로벌 | ❌ |
| Skin Bliss | ✅ AI 분석 | ✅ 기본 | ❌ 글로벌 | ✅ (14개 언어) |
| **BARDA** | **✅ 30종 가이드** | **✅ 15규칙 + 7일 캘린더** | **✅ 300개 K-뷰티** | **✅** |

> **BARDA의 포지션**: K-뷰티 제품 DB × 조합 충돌 분석 × 한국어 커뮤니티 = 겹치는 경쟁자 없음

## 1. 글로벌 벤치마킹 (상세)

### 1-1. FeelinMySkin — 가장 참고해야 할 모델 ⭐⭐⭐

부부 개발자 2명이 만든 인디 앱. BARDA와 가장 유사한 포지션이면서 "서비스 구색"이 가장 잘 갖춰져 있음.

| 항목 | 내용 |
|---|---|
| 운영사 | Andras Munkacsi (부부 개발, 헝가리) |
| 제품 DB | 15만개+ |
| 플랫폼 | iOS + Android + Chrome 확장 |
| 수익모델 | 프리미엄 구독 (PRO) |
| 핵심 강점 | "매일 돌아오게 만드는" 트래킹 구조 |

**기능 구성:**

| 기능 | 설명 | BARDA 대응 | 구현 상태 |
|---|---|---|---|
| 루틴 플래너 | AM/PM 루틴 설정 + Smart Ordering + 체크리스트 | 분석 → 체크리스트 전환 | ✅ 구현됨 (홈 화면) |
| 성분 분석기 | INCI 성분 체커 + 피부 고민별 매칭 점수 | 성분 가이드 30종 | ✅ 구현됨 (/guide) |
| 제품 트래커 | 보유 제품 목록 + 사용량 + 유통기한 | "내 서랍" 기능 | 🟡 Phase 3 예정 |
| 스킨 다이어리 | 매일 사진 + 수면/기분/운동 로그 + Analytics | 이모지 + 메모 다이어리 | ✅ 구현됨 (홈 화면) |
| 커뮤니티 포럼 | 질문/답변 + 루틴 공유 + Study Hub | 피드 + 댓글 | ✅ 구현됨 (/feed) |
| 제품 스캐너 | 사진 → DB 검색 + Skin Match Score | Phase 3 기능 | ❌ 미구현 |

**FeelinMySkin에서 배울 핵심:**
- "분석"은 입구이고, "매일 트래킹"이 서비스의 본체 → **BARDA: 체크리스트로 구현 완료**

- 교육 콘텐츠(Study Hub)가 신뢰감 + SEO 유입 → **BARDA: 성분 가이드 30종으로 대응**

## 5. BARDA만의 차별점 정리

### 경쟁자가 없는 영역 ✨

| 차별점 | 설명 | 경쟁 상황 |
|---|---|---|
| **K-뷰티 조합 분석** | 한국 제품 300개 × 15 충돌 규칙 × 한국어 | 글로벌 서비스는 한국 제품 미지원, 한국 서비스(화해)는 조합 분석 없음 |
| **루틴 점수 시스템** | 100점 만점 루틴 평가 | 어디에도 없는 BARDA 오리지널 |
| **7일 자동 캘린더** | 레티놀/각질 자동 분배 | 유일하게 자동화된 주간 스케줄링 |
| **AM/PM 맞춤 TIP** | 피부타입×시간대별 가이드 | 다른 앱에 없음 |
| **커뮤니티 + 분석 통합** | "이 루틴 따라하기" → 즉시 분석 | 분석과 커뮤니티가 분리된 타 서비스와 차별 |
| **원타임 결제 (₩9,900)** | 구독 피로도 없는 모델 | 대부분 구독제 |

### 경쟁 우위 공식:
```
BARDA = K-뷰티 특화 제품 DB
      × 조합 충돌 분석 (유일)
      × 매일 쓰는 트래킹 (체크리스트 + 다이어리)
      × 한국어 커뮤니티 (루틴 공유 + 따라하기)
```
> 이 4개가 겹치는 서비스가 현재 없음. **이 교집합이 BARDA의 해자(moat)**.
